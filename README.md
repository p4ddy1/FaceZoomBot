# FaceZoomBot

This is a Telegram Bot which you can send images of people and it will send only their enlarged faces back.
If you want to try it message: `@face_zoom_bot` on [Telegram](https://telegram.org). I created this bot to experiment
with building a distributed application. You can start multiple instances of this bot on many different systems, as long
as they can connect to the same RabbitMQ server and MongoDB/distributed filesystem.
